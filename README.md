## About

FlareHorz is a 2D game engine built using modular programming paradigm.

## Getting Started

### 1. Clone the Repository

```
git clone https://github.com/fnaxi/FlareHorz.git FlareHorz
cd FlareHorz
```

### 2. Run setup script

Run `GenerateProjectFiles.bat`

### 3. Open `FlareHorz.sln`

Open FlareHorz.sln and build all the projects.

## Code style

### Prefixes

- `C` - for classes.
- `S` - for structs.
- `E` - for enums and enum classes.
- `I` - for interfaces.
- `T` - for template types and params.
- `U` - for `using` types.
- `G` - for global variables.
- `b` - for boolean variables.
- `In` - for some in arguments.
- `Out` - for out arguments.
- `Z_` - for reflection or boilerplate code.
- `FH_` - for FlareHorz specific macros.
